Aidanna 🧠✨

An AI-powered task management app built with Expo (React Native) to help people organize, prioritize, and complete their daily goals smarter and faster.

🚧 Status

This project is currently in development. Features and designs are actively being built and refined.

🌟 Vision

Aidanna aims to be more than just a task list. With AI assistance, users will:

✅ Add and manage tasks easily

🤖 Get smart suggestions on priorities & deadlines

📅 Organize tasks into categories and projects

🔔 Receive reminders and productivity nudges

🌐 Sync across devices seamlessly

🛠️ Tech Stack

Frontend: Expo (React Native)

AI Integration: OpenAI API (planned)

Backend: Node.js / Express (planned)

Database: Supabase or Firebase (under review)
